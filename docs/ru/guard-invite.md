# Назначение охранника

Инструкция для **администратора чата**: как создать приглашение и назначить пользователя Telegram охранником для домового чата. Охранник сможет проверять пропуска автомобилей на КПП.

---

## Зачем назначать охранника

Охранник — это роль пользователя, который на пропускном пункте проверяет, разрешён ли въезд автомобилю. Чтобы пользователь получил возможность проверять пропуска в боте, администратор должен **создать приглашение** и отправить его будущему охраннику. После принятия приглашения пользователь получит сообщение «Вы назначены охранником» и сможет использовать [проверку пропусков](car-pass-check.md).

---

## Как создать приглашение для охранника

1. Откройте бота [@g00dwin_bot](https://t.me/g00dwin_bot) (в личных сообщениях или в чате).
2. Выполните команду **`/guard`** или отправьте сообщение **«Охрана»**.
3. Нажмите кнопку **«Создать приглашение»**.
4. Выберите **чат** (домовой чат или чат посёлка), к которому будет привязан охранник.
5. Бот создаст **одноразовую ссылку** на приглашение.

---

## Как отправить приглашение пользователю

- **Вариант 1:** Нажмите на созданную ссылку. Откроется интерфейс, где можно **выбрать пользователя Telegram** — ему будет отправлено приглашение стать охранником.
- **Вариант 2:** Скопируйте ссылку и отправьте её будущему охраннику вручную (в личку или в чат). Пользователь переходит по ссылке и принимает приглашение.

После перехода по ссылке и успешного принятия приглашения пользователь получит сообщение от бота: **«Вы назначены охранником»**. С этого момента он может проверять пропуска автомобилей для выбранного чата (дома/посёлка).

---

## Кратко: команды и кнопки

| Действие | Как сделать |
|----------|--------------|
| Открыть раздел «Охрана» | Команда **`/guard`** или сообщение **«Охрана»** |
| Создать приглашение | Кнопка **«Создать приглашение»** → выбрать чат → получить одноразовую ссылку |
| Назначить охранника | Перейти по ссылке → выбрать пользователя (или отправить ссылку ему) → пользователь переходит по ссылке и получает «Вы назначены охранником» |
