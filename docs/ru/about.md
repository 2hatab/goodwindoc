# О цифровом помощнике Goodwin

**Goodwin** — цифровой помощник с функциональностью чат-ботов для мессенджеров. Сейчас поддерживаются:

- **Telegram** — бот [@g00dwin_bot](https://t.me/g00dwin_bot) для групп и супергрупп
- **MAX** — поддержка в разработке

Бот помогает админам чатов управлять группами, автоматизировать рутину и улучшать общение участников.

## Для кого эта документация

- **Администраторы чатов** — быстрый старт по добавлению и настройке бота в группу
- **Пользователи** — описание команд и возможностей бота
