# About Goodwin Digital Assistant

**Goodwin** is a digital assistant with chatbot functionality for messengers. Currently supported:

- **Telegram** — bot [@g00dwin_bot](https://t.me/g00dwin_bot) for groups and supergroups
- **MAX** — support in development

The bot helps chat admins manage groups, automate routine tasks, and improve communication between members.

## Who this documentation is for

- **Chat administrators** — quick start for adding and configuring the bot in a group
- **Users** — description of bot commands and features
